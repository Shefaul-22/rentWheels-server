# 🚀 RentWheels – Backend Server

This is the backend server for **RentWheels**, an online car rental platform. It provides secure API endpoints for user management, car listings, and bookings. Built with **Node.js**, **Express.js**, **MongoDB**, and **Firebase Admin** for authentication.

---

## 🛠️ Tech Stack

- **Node.js & Express.js** – Server and routing
- **MongoDB** – Database
- **Firebase Admin** – User authentication verification
- **CORS** – Cross-Origin Resource Sharing
- **dotenv** – Environment variable management
